# 柬埔寨直升机/机票/旅游业务引流机器人

## 功能简介
- 自动欢迎语，所有消息都引导用户联系人工客服 @Boatbabes
- 不暴露机器人身份，所有回复均以客服口吻

## 使用方法
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 编辑 `bot.py`，将 `YOUR_TOKEN` 替换为你在 BotFather 获取的 Token。
3. 运行机器人：
   ```bash
   python bot.py
   ```

## 依赖
- python-telegram-bot==20.7

## 注意事项
- 建议将机器人描述、简介等设置为服务相关内容，提升搜索命中率。
- 机器人上线后，建议自己用小号测试体验。
